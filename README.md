# VisionTale Creator

VisionTale Creator is an AI-powered cinematic storytelling platform that transforms simple prompts into immersive scene-based visual narratives.

## Features

* AI story generation
* Genre-based storytelling
* Scene-by-scene cinematic breakdown
* AI-generated scene images
* Cyberpunk futuristic UI
* Responsive glassmorphism design
* Smooth loading animations
* Dynamic storytelling workflow

## Tech Stack

* React
* TypeScript
* Tailwind CSS
* TanStack Start
* Cloudflare Workers
* Pollinations AI

## Project Goal

To create a cinematic AI storytelling experience capable of generating immersive visual narratives from simple user prompts.

## Future Improvements

* Voice narration
* Video generation
* Story export
* AI storyboard creator
* Download functionality

## Status

Currently under active development.
